# Shop & Meals

A mobile-friendly shopping list, meal planner, and recipe app. Static single-page app — no build step.

- **Shop** — sectioned shopping list with check-off and progress, saved to `localStorage`.
- **Meals** — three-week dinner/lunch plan with protein tags.
- **Recipes** — expandable recipe cards with ingredients and method.

## Deploy

Static site served straight from `index.html`. Deployed on Vercel; any push to `main` redeploys automatically.
